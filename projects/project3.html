TicTacToe

The project was to simply create a web TicTacToe game using ReactJS. The project idea was taken from the official react website where a tutorial guides you through the process of making most of the game. There are some additional features such as displaying the specific row/column corresponding to the move made, displaying "draw" if the game ends in one, and others such as creating the board using loops.
